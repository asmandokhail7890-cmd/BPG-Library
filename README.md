# BPG-Library

## Balochistan Procurement Guide (BPG)

### Purpose

This repository is the official knowledge library of the Balochistan Procurement Guide (BPG). It contains all legal, regulatory, technical, and research documents related to public procurement.

---

# Current Scope

Currently covered:

* BPPRA

Future expansion:

* Federal PPRA
* Punjab PPRA
* Sindh PPRA (SPPRA)
* KPRA
* PEC
* NHA

# PRC Decision Rules

* Appeal Number is the primary identifier.
* Keep original PDF.
* Never rename Appeal Number.
* Consolidated appeals remain one PDF.
* Check duplicate before adding.
* Keep only official decisions.

Example:

PRC-045-2026_Mahd_Engineers_Pvt_Ltd.pdf

PRC-051-052-2026_Consolidated_Decision.pdf

---

# Duplicate Policy

Before adding any document:

* Check document number.
* Check title.
* Check date.
* Check version.
* Keep latest official version.
* Archive old versions if legally important.

---

# Working Rules

* Never change folder structure without approval.
* Never delete documents without verification.
* Organize first.
* Verify second.
* Rename third.
* Commit changes after verification.
